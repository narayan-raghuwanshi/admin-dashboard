Implementation of features like:
1. Pagination.
2. Search.
3. Multiselect.
4. Changes at place.


```bash 
git clone https://github.com/narayan-raghuwanshi/admin-dashboard
```
```bash
cd admin-dashboard
```
```bash
npm install
```
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
